# 📚 Python Web Scraper

![Python](https://img.shields.io/badge/Python-3.13-blue)

![License](https://img.shields.io/badge/License-MIT-yellow)

A Python project that scrapes book titles and prices from the **Books to Scrape** demo website and exports the results to a CSV file.

---

## 🚀 Features

- 📖 Scrapes book titles
- 💷 Extracts book prices
- 📄 Saves data to CSV
- ⚡ Fast and lightweight
- 🐍 Written in pure Python

---

## 🛠 Technologies

- Python
- Requests
- BeautifulSoup4
- Pandas

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/esemenec99-stack/python-web-scraper.git
```

Open the project:

```bash
cd python-web-scraper
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶ Run

```bash
python scraper.py
```
