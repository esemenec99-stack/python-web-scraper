# 📚 Python Web Scraper

A simple Python web scraper that collects book titles and prices from https://books.toscrape.com.

## Features

- Scrapes book titles
- Extracts book prices
- Saves data to CSV
- Uses BeautifulSoup and Pandas

## Technologies

- Python
- Requests
- BeautifulSoup
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python scraper.py
```

## Output

The program creates:

```
output.csv
```

Example:

| Title | Price |
|-------|-------|
| A Light in the Attic | £51.77 |
